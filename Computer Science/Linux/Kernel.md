A Kernel is the core of an [operating system](Operating Systems). 

It communicates between hardware and software. 
It provides low level access to memory, CPU and I/O. 
It is the first part of the operating system which is loaded into memory when the system is booted.

Provides access to 
- Hardware
- Software
	- Files and data
	- Programs
	- Loading programs into memory
	- Networks
	- Security and firewalls
	- Device driver management
	- etc

Users communicate to the kernel with a [shell](Shell.md).
